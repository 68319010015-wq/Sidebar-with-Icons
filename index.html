วิชาองค์ประกอบของคอมพิวเตอร์

ยินดีต้อนรับเข้าสู่บทเรียนออนไลน์!

เนื้อหาประกอบด้วยหน่วยการเรียนรู้เกี่ยวกับส่วนประกอบต่าง ๆ ของคอมพิวเตอร์และการทำงานของระบบคอมพิวเตอร์ เพื่อให้คุณเข้าใจสถาปัตยกรรมและการเชื่อมต่อของฮาร์ดแวร์แต่ละชิ้นได้อย่างลึกซึ้ง

CPU (Central Processing Unit)

หน่วยประมวลผลกลาง

เปรียบเสมือนสมองของคอมพิวเตอร์ ทำหน้าที่คำนวณ ตีความคำสั่ง และควบคุมการทำงานของฮาร์ดแวร์ส่วนอื่น ๆ ในระบบคอมพิวเตอร์

RAM (Random Access Memory)

หน่วยความจำหลักแบบชั่วคราว

ทำหน้าที่เก็บข้อมูลที่คอมพิวเตอร์กำลังใช้งานอยู่ในขณะนั้น เพื่อให้ CPU สามารถเข้าถึงข้อมูลได้อย่างรวดเร็ว เมื่อปิดเครื่องข้อมูลจะหายไป

Storage (หน่วยจัดเก็บข้อมูล)

หน่วยความจำสำรอง

ใช้สำหรับบันทึกข้อมูลและโปรแกรมต่าง ๆ ไว้ในเครื่องแบบถาวร แม้ปิดเครื่องข้อมูลก็ไม่สูญหาย เช่น SSD (Solid State Drive) และ HDD (Hard Disk Drive)

Output Devices (อุปกรณ์แสดงผล)

การแสดงผลลัพธ์ข้อมูล

ทำหน้าที่นำผลลัพธ์ที่ได้จากการประมวลผลมาแสดงให้อยู่ในรูปแบบที่มนุษย์เข้าใจ เช่น จอภาพ (Monitor), เครื่องพิมพ์ (Printer) และลำโพง (Speaker)

Input Devices (อุปกรณ์รับข้อมูล)

การนำเข้าข้อมูล

เป็นอุปกรณ์ที่ทำหน้าที่รับคำสั่งหรือข้อมูลจากผู้ใช้ส่งเข้าไปยังคอมพิวเตอร์ เช่น แป้นพิมพ์ (Keyboard), เมาส์ (Mouse) และไมโครโฟน (Microphone)

Computer Network (ระบบเครือข่าย)

การสื่อสารข้อมูล

การเชื่อมต่อคอมพิวเตอร์ตั้งแต่สองเครื่องขึ้นไปเข้าด้วยกัน เพื่อใช้ในการแลกเปลี่ยนข้อมูลข่าวสาร หรือใช้อุปกรณ์ฮาร์ดแวร์ร่วมกัน เช่น การใช้อินเทอร์เน็ต และเราเตอร์ (Router)
